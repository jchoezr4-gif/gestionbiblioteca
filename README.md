# Sistema de Gestión de Biblioteca
## Descripción del Proyecto
Sistema que permite registrar libros, gestionar préstamos, devoluciones y controlar la disponibilidad de ejemplares.

## Objetivos
- Organizar los libros de la biblioteca.
- Controlar préstamos y devoluciones.
- Permitir búsquedas rápidas.
- Mantener historial de uso.

## Reflexión sobre Control de Versiones
El uso de Git y GitHub permite:
- Mantener orden en la documentación.
- Historial claro de cambios (commits).
- Trabajar con versiones del mismo archivo.
- Evitar pérdida o confusión de documentos.
