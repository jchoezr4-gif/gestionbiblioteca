# Uso de Markdown

## ¿Qué es Markdown?
Lenguaje ligero para crear documentos rápidos y ordenados.

## ¿Por qué se usa?
- Fácil de leer
- Compatible con GitHub
- Ideal para documentación

## Ejemplo
# Título
## Subtítulo
- Lista
- Elemento

| Col | Col |
|-----|-----|
| A | B |

## Ventajas en GitHub
- Vista previa automática
- Permite versionado
- Ocupa poco espacio
