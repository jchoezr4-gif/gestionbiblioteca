# Propuesta de Mantenimiento – Sistema de Biblioteca

## Tipo de mantenimiento: Perfectivo

Este mantenimiento busca mejorar el rendimiento y usabilidad.

### Mejoras propuestas
- Implementar paginación para listas largas de libros.
- Mostrar historial de préstamos por usuario.
- Mejorar tiempos de búsqueda.
- Interfaz más intuitiva.
